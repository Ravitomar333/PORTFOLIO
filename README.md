# PORTFOLIO
A responsive personal portfolio website showcasing my skills, projects, and education. Built with HTML, CSS, and JavaScript to create a modern and user-friendly design.

## ✨ Features  
- Responsive design for all devices (mobile, tablet, desktop)  
- Smooth navigation with separate sections (About, Skills, Projects, Education, Contact)  
- Integration of technology logos for skills  
- Clean and minimal UI with focus on user experience

- ## 🛠️ Tech Stack  
- **Frontend:** HTML, CSS, JavaScript  
- **Version Control:** Git & GitHub
